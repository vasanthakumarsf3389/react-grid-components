# Component Review Prompt

This document outlines the steps for reviewing a component using the prompts defined in the review-prompt.md file.

## Review Process

1.Initial Prompt

Attach the guidelines and standard files from the Syncfusion Continuous AI tool. These documents will serve as the foundation for evaluating the component.

2.Source Code Prompt

Attach the component's source code. The review will be conducted based on the provided guidelines, component structure, and adherence to React standards.

3.Functionality Review Prompt

Request a functionality-based review of the component. This step ensures that the component behaves as expected and meets the required specifications.
